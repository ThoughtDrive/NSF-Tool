Description

	NSF Tool is a utility that can edit the song information contained 
	in NSF files. It can also use this information to batch rename your
	files according to a user defined pattern. Both NSF and the slightly
	newer NSFe formats are supported. 
	
	Note: NSF files contain music code from NES games. To listen to 
	this music you need some kind of NSF player. Visit the NSF Tool
	homepage for some starter links to get NSF music & players.
